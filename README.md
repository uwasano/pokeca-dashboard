# pokeca-dashboard(限定公開ビューア)

ポケカ利益検知システムのダッシュボード閲覧用ページ。

- `index.html` は **AES-256 で暗号化済み**(StatiCrypt)。中身はログインするまで復号されない。
- ログインは **メールアドレス + パスワード**(両方一致した場合のみ表示)。
- 表示されるデータは **合成サンプルのみ**。実データ・システム本体のコードは含まれない。
- システム本体(pokeca-profit)は別の非公開リポジトリで管理する。**このリポジトリに本体コードを置かないこと。**

## 認証情報の変更

閲覧用メール/パスワードを変えたい場合は `index.html` を作り直す必要がある
(暗号化し直し)。担当のClaudeセッションに依頼する。
