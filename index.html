ポケモンカード価格差・仕入れ可否 判断書(システム採用ルール)

■ 大原則
・ショップ販売価格を相場基準にしない。基準は「同状態の直近成約価格 > 複数店舗の買取 > オークション落札 > フリマ売り切れ > ショップ販売」。
・出品中価格は希望価格。相場は同一型番・言語・鑑定区分・近い状態で最低3件(可能なら5件)。1〜2件は参考値。
・PSA10と素体、異なる状態をまとめて中央値にしない。中央値優先(平均は外れ値に弱い)。相場より極端に安い専用/まとめ売り/状態難は外れ値除外。

■ 同一カード確認(必須一致)
カード名 / 型番 / 収録 / レアリティ / 言語 / 仕様(通常・ミラー・プロモ) / 1st有無 / 再録or初版 / 鑑定有無 / 鑑定会社 / グレード / エラー個体でないこと。

■ 100点採点(§27)
価格優位性25 / 状態25 / 真贋信頼性15 / 再販流動性15 / 出品者信頼性10 / 写真・説明信頼性10。
判定バンド: 85〜即検討 / 75〜有力候補 / 65〜条件付き / 55〜慎重 / 45〜見送り / 44以下 除外。
強制除外: 偽物疑い・型番不一致・写真転載・重大な折れ/水濡れ・安心鑑定拒否・利益マイナス・状態説明の重大矛盾・外部決済要求。

■ 必要利益率(§20)
即売型10%以上 / 通常転売15%以上 / 状態不明品20%以上 / 高額傷あり20〜25%以上 / 真贋リスクは原則除外。高額カードを数千円の利益で仕入れない。

■ 利益計算(§19)
純利益 = 想定売却価格 − 手数料10% − 送料 − 梱包 − 仕入 − 鑑定費 − その他。オークション途中価格・売り切れ表示を成約価格として断定しない。

■ 現物確認が必要(Claude API/人手)
表面/裏面の線傷・白欠け・凹み・折れ・水濡れ・センタリング・四隅四辺、斜光写真の有無、出品者の説明姿勢、査定/鑑定歴。価格計算では出せないため、上位候補のみ説明文・写真を精密判定する(損失回避・真贋・状態の確定)。
